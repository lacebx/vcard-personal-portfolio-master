# vCard, Personal Portfolio Website

My personal portfolio site, built as a vCard-style single-page application. Clean, dark, professional. It covers everything, who I am, what I've built, my resume, and how to reach me.

I designed and coded this myself: 750+ lines of HTML, a custom CSS stylesheet, and organized assets. Four branches of iteration, thirteen commits of refinement. This is the version I'm proud enough to share.

Your portfolio is the first thing people see and I think this is birarenze, so make it count.

---

## Sections

| Section | What's There |
|---------|---------------|
| **About** | Brief bio, what I'm doing (AI, cybersecurity, automation, consulting) |
| **Resume** | Education timeline and professional experience |
| **Projects** | Technical work: ML assistants, games, e-commerce, Web3, automation |
| **Contact** | Location map and contact form |

## Tech Stack

- **Markup:** HTML5 (750+ lines)
- **Styling:** Custom CSS (`assets/css/style.css`)
- **Assets:** SVG icons, images, custom favicon
- **Dev:** BrowserSync (`bs-config.json`) for live reload

---

## Running It
No build step needed. Open `index.html` in a browser or use BrowserSync for live reload:

```bash
npm install -g browser-sync
browser-sync start --config bs-config.json
```

## Why I Built This
I needed a portfolio that looked professional, loaded fast, and told my story clearly. No frameworks, no CMS, just HTML and CSS doing exactly what they were designed to do. If you're reading this on GitHub, go view it in a browser to see it properly.
